# 🌐 Network Connectivity & Troubleshooting Assignment

## 📌 Project Summary

This networking assignment demonstrates basic network troubleshooting and connectivity testing within a simulated organizational network environment. The project focused on testing communication between different departments using ICMP ping commands and identifying both successful and failed network connections.

The network included communication testing between:
- HR
- Admin
- Sales

During the lab, connectivity tests were performed to determine whether devices across departments could successfully communicate with each other.

### ✅ Successful Connectivity Test
A successful ping test was completed from the Sales department to Admin, confirming proper network communication between both systems.

### ❌ Failed Connectivity Test
A failed ping test was identified from HR to Admin, demonstrating an unreachable connection and highlighting potential network configuration or communication issues that would require troubleshooting.
